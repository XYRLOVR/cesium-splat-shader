🌍 Cesium照片地理标记系统

基于CesiumJS的3D地球可视化系统，支持照片地理标记、距离测量、模型漫游等功能。

📋 GitHub Pages 部署说明

此项目已配置自动部署到GitHub Pages。要在GitHub上运行：

  1. 将代码推送到GitHub仓库的main分支
  2. 在仓库设置中启用GitHub Pages
  3. GitHub Actions将自动构建和部署
  4. 访问 https://yourusername.github.io/repository-name/
📷 启动照片地理标记系统 (GitHub Pages版) 🌐 Hello World 示例

功能特色：

  • 📸 支持上传带GPS信息的照片并在3D地球上标记
  • 📏 距离测量工具
  • 🚶 3D场景自动漫游
  • 🗺️ 天地图底图集成
  • 🏫 南宁师范大学3D模型展示

技术栈：

  • CesiumJS 1.125.0 - 3D地球可视化引擎
  • @cesium/engine - Cesium核心引擎
  • @cesium/widgets - Cesium UI组件
  • 天地图API - 中国地图数据服务
